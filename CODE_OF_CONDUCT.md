# Code of Conduct

## Our pledge

We want participation in this project to be a safe, welcoming, and harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Welcome feedback and different viewpoints.
- Focus on what’s best for the community.

## Unacceptable behavior

- Harassment, discrimination, or hateful conduct.
- Personal attacks, trolling, or sustained disruption.
- Sexualized language or unwanted sexual attention.
- Publishing others’ private information (“doxxing”).
- Any other conduct that a reasonable person would consider inappropriate in a professional context.

## Scope

This code of conduct applies to all project spaces (issues, pull requests, discussions, and other official communication channels) and when representing the project in public.

## Reporting

If you experience or witness unacceptable behavior, please report it privately.

- Preferred: contact the maintainer via the email address listed on the GitHub profile for this repository.
- Alternative: if email is not possible, open an issue requesting a private contact channel (do not include sensitive details).

## Enforcement

Project maintainers may take any action they deem appropriate, including warnings, temporary bans, or permanent removal from the community spaces.

## Attribution

This policy is inspired by common open-source community guidelines (e.g., Contributor Covenant), but is tailored for this repository.
